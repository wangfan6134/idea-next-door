# 隔壁有个想法

这是「隔壁有个想法」的本地 VitePress 网站。

## 本地预览

```bash
cd /Users/wangfan/vitepress-site
npm install
npm run dev
```

浏览器打开：

```text
http://127.0.0.1:5173
```

## 添加文章

把 Markdown 文件放到：

```text
/Users/wangfan/vitepress-site/docs/articles/
```

然后在：

```text
/Users/wangfan/vitepress-site/docs/articles/index.md
```

添加文章链接。
