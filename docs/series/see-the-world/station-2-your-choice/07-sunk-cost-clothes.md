# 买了不穿的衣服为什么舍不得扔

**前情提要：** 上一篇说了贸易共赢，人和人交易能让双方都更好。今天聊聊为什么你明知道亏了却不愿止损。

---

打开衣柜，角落里躺着一件大衣，买的时候1200，吊牌都没拆，穿了零次。每次整理衣柜你都看见它，每次都说"下次穿"，下次永远是下次。扔？舍不得。送人？心疼。卖二手？50块都没人要。

这就是"沉没成本"。1200块已经花出去了，不管你扔不扔，那钱都回不来。它就像泼出去的水，你盯着地上的水看再久，杯子也不会自己满。经济学告诉你一个冷酷的规则：已经发生且无法收回的支出，不应该影响你接下来的决策。你该问的不是"这件衣服值不值1200"，而是"此刻这件衣服对我还有没有用"。没用就扔，就这么简单。但人不是机器，那1200像一根绳子拴着你，让你越拖越舍不得。

📌 **生活里还能看到这个**

电影看了一半发现是烂片，想着票都买了硬看完，结果赔了时间又赔了心情。

→ 谈了三年的恋爱明知道不合适，舍不得分
→ 考了个证发现没用，硬继续考
→ 沉没成本让你在每个该走的地方多待了太久
