# 你的经济学工具箱总复习

**前情提要：** 上一篇说了宏观经济学应用——你用学过的概念串成链，能看懂经济新闻了。这是最后一篇。我们来把六站走完的路重走一遍，看看你的工具箱里到底装了什么。

---

38篇之前，你连"机会成本"都没听过。第一次点开这个号，你可能只是想弄明白"为什么奶茶一直在涨价"。现在呢——你能跟同事聊通胀、跟房东掰扯租售比、面试时报对期望薪资、看到降准新闻知道房贷要变、纠结买不买房的时候不会再被"租金是打水漂"骗。

你不是变聪明了。你是多了一套工具箱。

来，打开看看里面有什么——

**第一站「价格的秘密」**（第1-8篇）：供需、弹性、价格管制。你学会了为什么春运火车票抢不到、为什么楼下水果店下雨天打折、为什么限价房总是买不到。

**第二站「消费者的选择」**（第9-14篇）：效用、预算约束、消费者剩余。你懂了为什么奶茶贵了你还买、为什么满减让你多花钱、为什么"买一送一"有时候并不划算。

**第三站「企业的逻辑」**（第15-20篇）：成本、利润、竞争。你能看穿公司为什么裁员、大促为什么打到骨折、咖啡店为什么越开越多。

**第四站「市场的失灵」**（第21-26篇）：外部性、公共品、信息不对称。你看清了二手车的坑在哪、为什么朋友圈代购的东西容易翻车、雾霾到底该谁买单。

**第五站「宏观大图景」**（第27-32篇）：GDP、通胀、失业、货币政策。你能读懂国家层面的经济运转，知道利率这个数字怎么影响你明天的房贷。

**第六站「你的工具箱」**（第33-38篇）：面试谈薪、买房租房、副业取舍、消费降级、新闻解读。你把前五站全装进了日常生活。

这一路上，你用奶茶和外卖学会了供需，用裁员和打折看清了企业逻辑，用二手交易和假疫苗读懂了市场失灵，用降准和CPI打通了宏观经济。你不是背了一堆名词，你是拿走了一整套**思考世界的方式**。

以后遇到任何经济问题，问自己三个问题就够了：谁在供给、谁在需求？真实的成本不只是钱还有什么？谁的利益被忽略了？

📌 **工具箱使用说明书**

供需看价格走向 → 机会成本看真实代价 → 比较优势看时间分配 → 效用最大化看消费 → 外部性看隐藏代价 → 宏观指标看趋势
