# 38篇走完你脑子里的世界变了

**前情提要：** 上一篇说了宏观经济学应用——今天说总结：38篇走完，你脑子里装了什么。

---

38篇之前，你连"机会成本"都没听过。第一次点开这个号，你可能只是想弄明白"为什么奶茶一直在涨价"。现在呢——你能跟同事聊通胀、跟房东掰扯租售比、面试时报对期望薪资、看到降准新闻知道房贷要变、纠结买不买房的时候不会再被"租金是打水漂"骗。

你不是变聪明了。你是多了一套工具箱。

来，打开看看里面有什么——

**第一站「地基」**（第1-6篇）：机会成本、边际效用、供需、贸易。你学会了为什么加班不如刷剧、为什么第三杯奶茶不如第一杯、为什么演唱会票永远抢不到。

**第二站「你的选择」**（第7-14篇）：沉没成本、心理账户、锚定效应、羊群效应、损失厌恶。你懂了为什么买了不穿的衣服舍不得扔、为什么花300吃饭不心疼运费心疼、为什么直播间里更容易下单。

**第三站「商家视角」**（第15-20篇）：成本定价、捆绑销售、价格歧视、信号理论、平台经济、声誉机制。你能看穿为什么喜茶涨价你还买、为什么机票你比别人贵、为什么网红店永远在排队。

**第四站「钱的旅行」**（第21-26篇）：GDP、通胀、利率、失业、货币供给、汇率。你读懂了工资涨了还是不够花、房贷利率变了意味着什么、买进口化妆品为什么贵了。

**第五站「世界失灵时」**（第27-32篇）：外部性、信息不对称、公共品、搭便车、囚徒困境、市场失灵与干预。你看清了邻居装修该不该赔、二手车市场的坑在哪、同事摸鱼你加班到底亏在哪。

**第六站「你的工具箱」**（第33-38篇）：面试谈薪、买房租房、副业取舍、消费降级、新闻解读。你把前五站全装进了日常生活。

这一路上，你用奶茶和外卖学会了供需，用裁员和打折看清了企业逻辑，用二手交易和假疫苗读懂了市场失灵，用降准和CPI打通了宏观经济。你不是背了一堆名词，你是拿走了一整套**思考世界的方式**。

以后遇到任何经济问题，问自己三个问题就够了：谁在供给、谁在需求？真实的成本不只是钱还有什么？谁的利益被忽略了？

📌 **工具箱使用说明书**

供需看价格走向 → 机会成本看真实代价 → 比较优势看时间分配 → 效用最大化看消费 → 外部性看隐藏代价 → 宏观指标看趋势
