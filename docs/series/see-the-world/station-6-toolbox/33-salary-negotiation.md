# 面试谈薪资的正确姿势

**前情提要：** 上一篇说了市场失灵与干预——政府什么时候该出手管一管。这一篇开始，我们把前面学的全装进工具箱，从最实际的开始：谈薪资。

---

HR微笑着问："你的期望薪资是多少？"你心里一紧——报高了怕被刷，报低了怕亏。大多数人最后憋出一句"按公司标准来"，然后回家后悔三天。

薪资谈判的本质，是一场**价格发现**。你在卖一种商品——你的劳动力。商品的价格不取决于你"觉得自己值多少"，而取决于市场上干这个活的有多少人、企业对这个岗位的需求有多少。供大于求，价格被往下压；供不应求，价格往上抬。

HR问你期望薪资，不是考你胆量，是试探你对自己市场价的认知准不准。报得离谱高，说明你不了解行情。报得离谱低，说明你好拿捏。两边都精准，说明你做了功课、懂规则。

所以面试前最重要的事不是练自我介绍，是先去招聘网站搜同岗位、同城市、同年限的薪资区间。那个中位数，就是你的市场价。报的时候取区间上限，给对方还价空间，也给自己留余地。你没有"值多少钱"，只有市场愿意为你付多少钱。

📌 **生活里还能看到这个**

不只是面试——跳槽谈薪、自由职业报价、接私活定价，本质都一样：你得先知道市场价，才有谈的底气。不知道行情的人只能被动接受；知道行情的人有筹码博弈。

跳槽前查三个平台薪资数据 → 面时报区间上限 → 给对方还价空间 → 最终落中上
