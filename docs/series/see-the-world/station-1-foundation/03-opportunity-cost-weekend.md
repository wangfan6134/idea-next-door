# 刷剧还是加班？选错亏大了

上一篇说了资源永远稀缺，那稀缺之后呢？就要做选择——今天说选择的隐形成本。

上周六你躺着刷了一天剧。

很爽对吧？

但经济学家会说：你其实花了钱，只是你没意识到。

**机会成本**：你做了A，就放弃了B。放弃的B，就是A的真实成本。

刷剧放弃了什么？
- 放弃了学新技能的下午
- 放弃了去拜访老朋友
- 放弃了做一顿好饭

这些被放弃的选项里，最有价值的那个，就是你刷剧的"机会成本"。

它不会出现在你的银行账单里，但它真实存在。

📌 生活里还能看到这个

你大学选了经济学，而不是计算机。

机会成本就是：你放弃的那条计算机人生里可能赚到的钱和经历。

所以大学报志愿的时候，不只是在选专业——是在选你愿意放弃什么。

没有好选择，只有你认为值得的选择。

> **Zero说：** 做选择从来不是"选了什么"，是"放弃了什么还觉得值"。
