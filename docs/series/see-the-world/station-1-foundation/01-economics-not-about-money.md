# 经济学不是教你赚钱的

经济学老师都没跟你说过一件事：

**经济学从来不教你怎么赚钱。**

你可能以为经济学 = 股市 + 基金 + 怎么让钱生钱。

错了。那叫金融学，或者个人理财。

经济学研究的是：**人在资源有限的情况下，怎么做决定。**

就这一句话。

你上午刷了两小时剧没去健身，这是经济学。
你双十一买了一堆用不上的东西，这是经济学。
你工资涨了但还是觉得不够花，这也是经济学。

跟K线图没关系。跟你**每一天的每一个选择**有关系。

📌 再想深一层

你妈问你"学经济学有什么用"，正确答案不是"能炒股"——

是：**能看懂这个世界为什么这样运转，然后少做蠢决定。**

这38篇，就是帮你建这个眼光的。从零开始，不跳步。

> **Zero说：** 经济学不教你赚钱，它教你看清楚你在哪里。

👋 你是从哪看到的这个号？你之前以为经济学是什么？
