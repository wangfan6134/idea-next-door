# 化工厂建在你家旁边你找谁说理

**前情提要：** 上一篇说了囚徒困境——今天说市场失灵与干预：市场搞不定时该靠谁。

---

你住的小区旁边要建一个化工厂，全楼住戶都反对。你去找开发商理论，开发商说手续齐全，环评已过。你找市场讲道理？市场不跟你聊天。你上二手平台搜索"阻止化工厂施工"，没有店铺。这时候你只能在业主群里接龙写联名信，一个电话打到环保局。这一刻你才意识到：**有些事，只有那个你觉得"最好别来管我"的角色才能管。**

这就是"市场失灵"——前面五篇讲的外部性、信息不对称、公共品、搭便车、囚徒困境，全是市场自己修不好的Bug。市场擅长让买卖双方一拍即合，但遇到第三方受伤、信息不对等、收费困难、人人搭车、互相不信任这些问题，就彻底失灵了。这时候政府出来补位：收排污税解决外部性，强制产品信息披露解决信息不对称，用财政修路建灯，定规则堵搭便车的漏洞，反垄断打破囚徒困境。

📌 **生活里还能看到这个**

你买的食品包装上有生产日期和配料表，不是厂家自己善良，是政府用法律强制要求的。你手机上的App有隐私协议，也不是平台主动想保护你，是监管在背后推了一把。你交的个人所得税，一部分就是你的"公共服务账单"。政府干预不是"管你"，是补市场管不了的窟窿。但尺子不能只有一把——管少了失灵，管多了也可能失灵。

→ 最低工资标准防止劳动力市场压低工资
→ 烟草税让抽烟者自己承担未来的医疗成本
→ 反垄断法防止一家独大收割消费者
