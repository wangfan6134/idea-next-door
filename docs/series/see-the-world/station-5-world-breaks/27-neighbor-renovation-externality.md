# 邻居装修吵到你该赔吗

**前情提要：** 上一篇说了汇率，进口化妆品涨价不只是汇率的事——从这篇开始，进入第五站，聊聊"市场也会出错"的那些事。

---

周末早上八点，你被楼上电钻声活活塞醒。加班熬了一整周就想睡个懒觉，结果邻居装修师傅准时开工。你躺在床上气得发抖，翻遍物业合同也找不到"禁止周末装修"这一条。人家花钱装修自己的房子，你有什么资格不让？

这就是经济学说的"外部性"——一笔交易里，买卖双方谈好了价钱，第三方却被迫承担了成本。邻居花自己的钱装修自己的房子，他和装修队的交易里没有你。但电钻的噪音、粉尘、震动，全算在了你头上。你既不是买家也不是卖家，却成了这笔交易最大的受害者。

经济学家把这种"别人做事你遭殃"叫做负外部性。反过来也有正外部性——你在阳台上种满花，路过的邻居白看了，你也没法找他们收门票钱。但真正让人头大的是负外部性：因为它意味着**私人做决定，社会付成本**。

📌 **生活里还能看到这个**

工厂排污是最典型的负外部性。工厂生产便宜的商品，消费者买到低价货，工厂赚了利润——三方里两方都开心。但河流被污染了，下游居民买净水器、生病吃药的钱，工厂一分不出。市场交易只算了买卖两方的账，第三方的损失被彻底忽略了。

→ 你不抽烟但室友在屋里抽，二手烟全归你
→ 楼下烧烤店油烟每天准时飘进你家窗户
→ 同事感冒还来公司，你第二天也开始打喷嚏
