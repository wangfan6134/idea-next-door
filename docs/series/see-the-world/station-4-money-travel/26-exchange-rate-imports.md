# 买进口化妆品为什么贵了

**前情提要：** 上一篇说了货币供给——国家印钱跟你的存款有什么关系。今天第四站收尾，说说汇率——为什么你买进口化妆品越来越贵。

---

你一直回购的那款精华，去年双十一480，今年618一看560。你气得去翻品牌官网，发现人家海外美元定价根本没变。你发小红书骂品牌割韭菜——但这次你真的骂错人了。

汇率就是两种钱之间的"换算比例"。比如现在1美元换7块人民币，如果明天变成1美元换7.3块，那就是人民币贬值了——你手里的人民币在国际市场上"缩水"了。进口化妆品以美元或欧元定价，人民币一贬值，同样的精华进了国内就要花更多人民币。品牌没涨价，是你的人民币在国际上不值钱了。你多掏的80块，不是给了品牌公关部，是给了汇率差。

📌 **生活里还能看到这个**

你计划去日本扫货，出发前一看汇率——去年100日元换5块人民币，今年变5.5了。同样的预算，能在当地买的东西变少了，还没出门就先亏了一笔。

→ 海淘同一件衣服，货价没变但到手的运费+税费贵了
→ 留学生的家长突然发现，学费一年多了好几万人民币
→ 代购跟你说"最近汇率不好要涨价"，不是借口，是真的

---

> **Zero说：** 汇率是你钱包的隐形汇率表。
