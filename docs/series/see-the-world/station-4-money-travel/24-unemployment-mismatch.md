# 为什么找不到工作却到处在招人

**前情提要：** 上一篇说了利率——今天说结构性失业：到处在招人，却找不到工作。

---

你已经投了200份简历，面试了三家，全没了下文。打开招聘App一看，满屏都是"急招""月入过万""五险一金"。你忍不住截图发姐妹群："这世界是不是有病？"

这叫结构性失业——不是没有岗位，是你的技能和岗位"对不上"。打个比方：一条街上奶茶店在急招调茶师，月薪七千；隔壁培训机构急招Python讲师，时薪两百。而你，做了五年行政。你能去调茶吗？理论上能，但要重新学，还得从小工做起；你能去教Python吗？理论上也能，但那得先学半年。岗位在，人也在，但就是配对失败。不是你不努力，是产业结构变了，你的技能标签还没更新。

📌 **生活里还能看到这个**

你去面试新媒体运营，对方说"要会拍剪、会写脚本、会投流，最好还会AI做图"。你每条都沾但每条都不精，对方礼貌地说了句"回去等通知"。

→ 35岁的项目经理被裁，招聘全写"要求28岁以下"
→ 你会做的事AI两秒钟能搞定，需要人做的事你学不会
→ 越来越多的人去考编考公，因为私企要的技能你"对不上"了
