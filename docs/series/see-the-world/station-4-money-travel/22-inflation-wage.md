# 工资涨了为什么还是不够花

**前情提要：** 上一篇说了GDP——国家有钱跟你有什么关系。今天说说为什么钱明明变多了，反而不够花。

---

年底老板心情好，给你涨了500块工资，你高兴得连点三天奶茶庆祝。结果到下个月你发现：奶茶从15涨到18，外卖配送费从3块涨到5块，就连常去那家早餐店的包子也从2块5涨到了3块。算下来，500块连塞牙缝都不够。

这叫通货膨胀——所有东西都在涨价，但你的钱不值钱了。打个比方：你们公司茶水间一共放着10个苹果和100块零钱，大家默认一个苹果10块。突然行政又放了100块进去，现在茶水间有200块了，但苹果还是10个。猜怎么着？大家开始抢，一个苹果涨到20块。钱多了，东西没多，价格就涨了。你的工资涨了500，但奶茶涨了3块、车费涨了、房租涨了……物价涨得比工资快，你实际能买到的东西反而更少了。

📌 **生活里还能看到这个**

你妈总说"以前一碗面才三块钱"，不是面变得多高级了，是钱变薄了。同样的配方同样的碗，四十年前卖三块现在卖十八块，中间差的十五块全是被通胀吃掉的。

→ 同一款粉底液去年280今年320，配方没变只是钱薄了
→ 房租每年续约涨5%，不是房东越来越贪，是钱在贬值
→ 你爸妈存了一辈子定期，现在发现利息跑不过菜价涨幅

---
